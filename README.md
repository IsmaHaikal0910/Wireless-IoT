## Overview

A collection of hands-on wireless and IoT engineering projects covering antenna simulation, real-time speed measurement, and WiFi signal characterization. Each project combines hardware experimentation with software-driven analysis and documentation.

---

## Projects

###  [EM Simulation of Antenna Characteristics](Wireless IoT/EM Simulation of Antenna Characteristics)

Electromagnetic simulation and analysis of antenna behavior using computational tools.

**Key topics:**
- Antenna radiation pattern modeling
- Gain, directivity, and impedance matching analysis
- Near-field and far-field EM simulation
- Parameter sweeps and frequency response characterization

**Tools & Technologies:** POSTFEKO / CADFEKO / MATLAB

---

###  Speed Measurement

Real-time speed measurement system using wireless/sensor-based techniques.

**Key topics:**
- Sensor integration and signal conditioning
- Real-time data acquisition and processing
- Wireless data transmission
- Calibration and measurement validation

**Tools & Technologies:** ESP32 · C++ · Sensor modules

---

###  WiFi Strength Measurement / Documentation

Systematic measurement and documentation of WiFi signal strength across environments.

**Key topics:**
- RSSI data collection and logging
- Spatial signal mapping
- Environmental interference analysis
- Structured technical documentation

**Tools & Technologies:** Python / ESP32 · WiFi modules · Data logging
