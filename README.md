## 🇬🇧 English

### Overview

A collection of hands-on wireless and IoT engineering projects covering antenna simulation, real-time speed measurement, and WiFi signal characterization. Each project combines hardware experimentation with software-driven analysis and documentation.

---

### Projects

####  [EM Simulation of Antenna Characteristics](https://github.com/IsmaHaikal0910/Wireless-IoT/tree/main/Projects/EM%20Simulation%20of%20Antenna%20Characteristics)

Electromagnetic simulation and analysis of antenna behavior using computational tools.

**Key topics:**
- Antenna radiation pattern modeling
- Gain, directivity, and impedance matching analysis
- Near-field and far-field EM simulation
- Parameter sweeps and frequency response characterization

**Tools & Technologies:** POSTFEKO / CADFEKO / MATLAB

---

####  [Speed Measurement](./Speed-Measurement)

Real-time speed measurement system using wireless/sensor-based techniques.

**Key topics:**
- Sensor integration and signal conditioning
- Real-time data acquisition and processing
- Wireless data transmission
- Calibration and measurement validation

**Tools & Technologies:** ESP32 · C++ · Sensor modules

---

####  [WiFi Strength Measurement / Documentation](./Wifi-Strength-Measurement)

Systematic measurement and documentation of WiFi signal strength across environments.

**Key topics:**
- RSSI data collection and logging
- Spatial signal mapping
- Environmental interference analysis
- Structured technical documentation

**Tools & Technologies:** Python / ESP32 · WiFi modules · Data logging

<a name="deutsch"></a>
## 🇩🇪 Deutsch

### Übersicht

Eine Sammlung praxisorientierter Wireless- und IoT-Ingenieurprojekte aus den Bereichen Antennensimulation, Echtzeit-Geschwindigkeitsmessung und WLAN-Signalcharakterisierung. Jedes Projekt verbindet Hardware-Experimente mit softwaregestützter Analyse und technischer Dokumentation.

---

### Projekte

####  [EM-Simulation von Antenneneigenschaften](./EM-Simulation-of-Antenna-Characteristics)

Elektromagnetische Simulation und Analyse des Antennenverhaltens mithilfe numerischer Berechnungswerkzeuge.

**Schwerpunkte:**
- Modellierung von Antennenstrahlungsdiagrammen
- Analyse von Gewinn, Richtwirkung und Impedanzanpassung
- Nah- und Fernfeld-EM-Simulation
- Parameterstudien und Frequenzgangcharakterisierung

**Werkzeuge & Technologien:** POSTFEKO / CADFEKO / MATLAB

---

####  [Geschwindigkeitsmessung](./Speed-Measurement)

Echtzeit-Geschwindigkeitsmesssystem auf Basis drahtloser und sensorgestützter Messtechnik.

**Schwerpunkte:**
- Sensorintegration und Signalaufbereitung
- Echtzeit-Datenerfassung und -verarbeitung
- Drahtlose Datenübertragung
- Kalibrierung und Messvalidierung

**Werkzeuge & Technologien:** ESP32 · C++ · Sensormodule

---

####  [WLAN-Signalstärkemessung / Dokumentation](./Wifi-Strength-Measurement)

Systematische Messung und Dokumentation der WLAN-Signalstärke in verschiedenen Umgebungen.

**Schwerpunkte:**
- RSSI-Datenerfassung und -protokollierung
- Räumliche Signalkartierung
- Analyse von Umgebungseinflüssen und Interferenzen
- Strukturierte technische Dokumentation

**Werkzeuge & Technologien:** Python / ESP32 · WLAN-Module · Datenlogging
